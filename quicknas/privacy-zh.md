# QuickNAS 隐私政策

最后更新：2026年2月13日

本隐私政策说明当你在 macOS 上使用 QuickNAS 应用时，QuickNAS（“我们”）如何处理相关信息。

## 1. 适用范围

本政策适用于 QuickNAS 应用的使用，以及通过 `support@unko.fun` 进行的支持沟通。

本政策不涵盖第三方服务（例如 Synology 服务、Apple 服务或你的邮件服务提供商）；这些服务受其各自政策约束。

## 2. 我们处理的数据

QuickNAS 主要设计为在你的设备本地运行。根据你使用的功能，应用可能处理以下数据：

### A. 账户与连接设置

- 显示名称
- 用户名
- 连接方式（Direct URL 或 QuickConnect）
- 端点地址 / QuickConnect ID
- 连接安全偏好（例如允许不安全连接）

### B. 会话凭据

- NAS 登录 API 返回的会话标识符/令牌
- 当应用行为启用会话持久化时，会存储在 macOS Keychain 中

### C. 传输与设置数据

- 传输历史元数据（文件名、状态、时间戳、进度、可选的分享链接/错误信息）
- 应用设置（上传路径规则、重名文件处理、语言/外观、通知偏好、打包大小限制、引导完成状态）

### D. 本地诊断数据

- 近期客户端日志（连接/会话/传输相关）
- 在设置中展示的应用性能/缓存指标

### E. 可选支持包（仅当你选择报告 Bug 时）

当你触发“Report Bug”时，QuickNAS 可能会先生成邮件草稿，并附加诊断文件供你确认，例如：

- 应用版本/构建号和 Bundle 标识符
- macOS 版本和区域设置
- 语言设置
- 当前账户显示名称与脱敏后的端点摘要
- 连接状态
- 近期客户端日志

是否发送邮件和附件由你决定。

## 3. 我们不会有意作为云服务收集的内容

QuickNAS 不提供核心功能使用所必需的 QuickNAS 云账户服务。

我们不会在应用内进行内置广告画像或行为广告投放。

## 4. 我们如何使用数据

我们处理数据用于：

- 与你的 NAS 建立连接并进行认证
- 上传文件并执行传输流程
- 应用你的偏好并将文件路由到已配置路径
- 展示传输状态、连接状态和存储概览
- 在你请求支持时排查问题
- 维护应用安全性与可靠性

## 5. 数据共享

我们仅在你触发的应用功能所需范围内，与第三方共享/处理数据：

- 你配置的 Synology/NAS 端点（包括你使用时的 QuickConnect）
- Apple 系统服务（例如 Keychain、通知框架、分享/邮件撰写服务）
- 你发送支持邮件时使用的邮件客户端/服务商

我们不会出售你的个人数据。

## 6. 数据保留

数据保留通常由本地应用状态和你的操作决定：

- 账户设置与传输历史会保留，直到你编辑或删除
- 会话凭据会保留在 Keychain 中，直到被替换、失效或移除
- 客户端日志会在本地保留，直到应用清理、轮转或你手动清除
- 你可以在应用中清除日志/缓存并移除账户

## 7. 安全性

在应用上下文中，我们采取合理的技术措施，包括使用 macOS Keychain 存储会话凭据。

任何传输或存储方式都无法保证 100% 安全。你有责任保护自己的设备和 NAS 环境安全。

## 8. 跨境处理

由于你配置的 NAS 或支持邮件服务提供商可能位于不同地区，数据可能在多个司法辖区内处理。

## 9. 你的选择

你可以：

- 随时停止使用应用
- 移除账户并清除本地日志/缓存
- 自行决定是否发送支持邮件/附件
- 在可用范围内配置连接行为，包括安全/不安全访问选项

## 10. 儿童

QuickNAS 不面向未达到你所在地区适用法律规定年龄的儿童。

## 11. 政策变更

我们可能会不时更新本隐私政策。“最后更新”日期反映最近一次修订时间。
